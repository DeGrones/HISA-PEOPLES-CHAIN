## What We Do

**HISA PEOPLES CHAIN tokenizes conservation methodologies** — transforming everyday forest actions into verifiable, tradeable digital assets on Hedera. We enable Kenyan Community Forest Associations (CFAs) and Community-Based Organizations (CBOs) to own, measure, and monetize their environmental stewardship through blockchain-verified impact.

By leveraging **Hedera Guardian** as the foundational policy-driven verification framework, the platform codifies conservation practices — tree planting, nursery management, agroforestry, and biodiversity monitoring — into immutable, market-ready records. Each verified action mints **JANI tokens**, creating a conservation finance value layer where forest stewards capture maximum economic benefit from their work.

This is not just a carbon credit platform. This is **community ownership of the value chain**.

## Core Innovation: Methodology Tokenization

### How It Works

```
┌──────────────────────────────────────────────────────────────────┐
│               CONSERVATION METHODOLOGY                           │
│   (e.g. Tree Planting Protocol v1.0 — defined in Guardian)      │
└──────────────────────────────────────────────────────────────────┘
                             │
                             ▼
        STEP 1 — ACTION
        CFA/CBO member performs conservation activity
        (GPS-tagged tree planting, nursery work, etc.)
                             │
                             ▼
        STEP 2 — VERIFICATION (Multi-Party)
        • Community validators on-site
        • AI image recognition confirms species & growth
        • Hedera Guardian enforces methodology policy rules
                             │
                             ▼
        STEP 3 — TIMESTAMPING (Hedera Consensus Service)
        Immutable record: treeId, species, GPS, validator IDs,
        timestamp, photo hash — permanently verifiable
                             │
                             ▼
        STEP 4 — TOKENIZATION (Hedera Token Service)
        1 JANI token minted to planter's wallet
        1 JANI = 10 kg CO₂ offset (annual, verifiable)
                             │
                             ▼
        STEP 5 — VALUE CREATION
        • Stake JANI for 5–15% APY yields
        • Sell carbon credits on global markets
        • Access UMOJA DeFi (loans, savings, insurance)
        • Participate in community DAO governance
```

### Verified Methodologies & Rewards

| Methodology | Description | Tokenization Outcome |
|-------------|-------------|----------------------|

| Nursery Management | Seedling propagation and survival monitoring | Tiered JANI based on survival rate |

Hedera serves as the **trust layer of the internet** underpinning all methodology tokenization.

|----------------|-------------|

## Getting Started

### Prerequisites

- Node.js v18+
- npm or yarn
- Hedera testnet account — [portal.hedera.com](https://portal.hedera.com)
- HashPack wallet for testing

### 1. Clone the Repository

```bash
git clone https://github.com/DeGrones/HISA-PEOPLES-CHAIN.git
cd HISA-PEOPLES-CHAIN
```

### 2. Backend Setup (`hisa-backend`)

```bash
cd hisa-backend
npm install
cp .env.example .env
# Fill in your Hedera credentials, MongoDB URI, and API keys
```

```bash
# Development mode (auto-restart via nodemon)
npm run dev

# Production mode
npm start
```

### 3. Frontend Setup (`front end/hisa-Dapp`)

```bash
cd "front end/hisa-Dapp"
npm install
cp .env.example .env
# Add RPC endpoints and backend URL
```

```bash
# Development mode (runs on port 9002)
npm run dev

# Optional: Genkit AI development server
npm run genkit:dev

# Production build
npm run build && npm start
```

### 4. Running Both Together

Open two terminal windows:

```bash
# Terminal 1 — Backend
cd hisa-backend && npm run dev

# Terminal 2 — Frontend
cd "front end/hisa-Dapp" && npm run dev
```

Then open `http://localhost:9002` in your browser.

### Environment Variables

```env
# hisa-backend/.env
HEDERA_NETWORK=testnet
MY_ACCOUNT_ID=0.0.xxxxx
MY_PRIVATE_KEY=302e0201...
MONGODB_URI=your_mongo_uri
```

### Hosted Version (No Setup Required)

🌐 [hisa-chain-dapp.vercel.app](https://hisa-chain-dapp.vercel.app/)

### Common Issues

- **Port conflict on 9002** — Next.js will prompt to use another port, or update the script in `package.json`
- **Backend won't start** — Check your `.env` database connection string
